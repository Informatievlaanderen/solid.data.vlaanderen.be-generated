An Access Request or an Access Grant.
Eigenschappen
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: for personal data.
Eigenschap Verwacht Type Kardinaliteit Beschrijving Gebruik Codelijst
for personal data Verifiable Credential The Verifiable Credential for/to which access is requested/granted.

Verifiable Credential

Beschrijving
Een verifieerbare credential is een fraudebestendige credential met auteurschap dat cryptografisch kan worden geverifieerd. Verifieerbare credentials kunnen worden gebruikt om verifieerbare presentaties te maken, die ook cryptografisch kunnen worden geverifieerd. De claims in een credential kunnen over verschillende onderwerpen gaan.
Eigenschappen
Voor deze entiteit zijn geen eigenschappen gedefinieerd.
